see on the internet. On the top part of the form there is a searchbar. Its function is to accept
user input to be searched for in a database. Second, there are txtfields that asks for yout personal
information (fullname, desired username, desired password, birthdate, and gender) which is necessary
for acquiring an account. The user can also choose to register as a student or as a teacher, and on the
bottom part of the form there is a checkbox which lets the user confirm with the terms and conditions of
the site they are singning uplastly, There are two buttons provided for submitting and resetting
